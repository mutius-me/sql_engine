# SQL Engine

## To run as a binary:
- Navigate to `sql_engine/dist/`.
- Run `./main <file_path_here.json>`.
- Pass in SQL queries as standard input.


## To run as a Python script:
- Install `uv` using `pip install uv`.
- Run `uv sync`.
- Run `p main.py <file_path_here.json>`.
- Pass in SQL queries as standard input.
